# Gerador de Senha

Este é um projeto simples em Python para gerar senhas aleatórias seguras, utilizando uma combinação de letras, números e símbolos.

## Descrição

O script `GeradorSenha.py` solicita ao usuário o número de caracteres desejado e gera uma senha aleatória com base nesse tamanho. A senha inclui letras maiúsculas e minúsculas, dígitos e pontuações para maior segurança.

## Requisitos

- Python 3.x instalado no sistema

## Como executar

1. Certifique-se de que o Python está instalado.
2. Execute o script no terminal:

   ```bash
   python GeradorSenha.py
   ```

3. Digite o número de caracteres desejado (deve ser maior que 0).
4. O programa exibirá a senha gerada.

## Exemplo de uso

```
==== Gerador de senha aleátorio ====
Qauntos caracteres você quer na senha?  12
Sua senha aleatória é:  A!b3D$5fG7hJ9
```

## Observações

- As senhas geradas são aleatórias e podem incluir caracteres especiais.
- Este é um gerador básico; para uso em produção, considere bibliotecas mais robustas como `secrets`.
